## 5. 关键 Bug 修复

### edge_attr 传递修复（vision_2d.py _predict_probs_from_doc）

**问题**：`_predict_probs_from_doc()` 中 `return_edge_attr` 检查仅匹配 `"edge_sage"`，但 v4 模型的 `model_type` 是 `"edge_sage_v2"`，导致 `edge_attr=None` 传入 GraphEncoderV2WithHead，触发 `edge_attr required` 错误。

**修复**：
```python
# 修复前
return_edge_attr=self.model_type == "edge_sage"

# 修复后
_needs_edge_attr = self.model_type in ("edge_sage", "edge_sage_v2")
return_edge_attr=_needs_edge_attr
```
