## 5. 共现关键词（B5.7b）

### 5.1 实现

在 `TextContentClassifier` 中新增 `COOCCURRENCE` 字典，当多个条件词同时出现时给予分数奖励：

```python
COOCCURRENCE = {
    "法兰": {
        "min_hits": 2,  # 至少 2 个条件词同时出现
        "keywords": ["密封面粗糙度", "RF面", "FF面", "突面", "凹凸面",
                     "螺栓孔圆", "法兰厚度", "PN16", "PN25", "PN40",
                     "NB/T", "HG/T", "GB/T 9119", "GB/T 9115"],
        "bonus": 0.15,  # 奖励系数
    },
    "箱体": {
        "min_hits": 2,
        "keywords": ["箱盖", "箱座", "窥视孔", "放油孔", "通气孔",
                     "非加工面涂漆", "迷宫密封"],
        "bonus": 0.12,
    },
}
```

### 5.2 验证结果

```
法兰 co-occurrence（密封面粗糙度+RF面+PN25+螺栓孔圆，无"法兰"词）→ 法兰=1.000 ✓
箱体 co-occurrence（箱盖+窥视孔+放油孔+通气孔）→ 箱体=1.000 ✓
单独 RF面 → {}（min_hits=2 未满足，正确放弃）✓
```

**关键改善**：即使 DXF 中不含"法兰"二字，只要有"密封面粗糙度"+"PN25"等组合出现，也能正确识别为法兰。
