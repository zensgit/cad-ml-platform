## 2. 从头蒸馏实验 v5c（运行中）

### 2.1 与 v5/v5b 的关键区别

| 参数 | v5 | v5b | **v5c** |
|------|-----|------|--------|
| 初始化 | v4 warm-start | v4 warm-start | **随机初始化** |
| alpha | 0.3 | 0.7 | **0.5** |
| temperature | 3.0 | 1.5 | **2.0** |
| lr | 5e-4 | 1e-4 | **5e-4** |
| 结果 | 91.0% | 90.6% | **训练中** |

### 2.2 为什么从头蒸馏可能有效

v5/v5b 失败因为 v4 的 warm-start 权重已固化了决策边界，soft label 
信号反而将边界推向次优位置。从头训练的学生没有预设偏见，能自由学习
教师的完整概率分布，包括类间相似度关系。
