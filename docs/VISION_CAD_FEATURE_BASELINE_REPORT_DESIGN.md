# Vision CAD Feature Baseline Report Design

## Overview
Provide a lightweight markdown report generator that summarizes benchmark
baseline JSON results into per-combo summary tables.

## Script
- `scripts/vision_cad_feature_baseline_report.py`

## CLI
- `--input-json`: benchmark JSON with a `results` list.
- `--output-md`: optional markdown output file.

## Output
- Markdown report with thresholds and summary metrics per combo.
