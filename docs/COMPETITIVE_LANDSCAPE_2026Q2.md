# CAD ML Platform - 同业竞品分析报告

**编制日期**: 2026-04-09
**分析范围**: 全球 CAD + AI/ML 分析平台
**分析目的**: 明确市场定位、识别竞争威胁、指导产品演进

---

## 一、市场概述

CAD + AI 市场正处于快速增长期，主要分为三个赛道：

| 赛道 | 代表厂商 | 核心价值 | 市场规模趋势 |
|------|---------|---------|-------------|
| **AI 辅助设计** | Autodesk, Siemens, Dassault, PTC | 帮设计师"画图更快" | 最大，巨头主导 |
| **几何搜索/查重** | CADENAS, SIMUFORM, Physna, simus | 帮企业"减少重复造轮子" | 中等，德系领先 |
| **智能图纸分析** | **CAD ML Platform (我们)**, HOOPS AI | 帮制造业"理解图纸内容" | 新兴，竞争少 |

**我们的赛道是第三个 —— 智能图纸分析微服务。** 这是一个新兴市场，直接竞品极少，但上下游巨头可能向此扩展。

## 二、竞品详细分析

### 2.1 最直接竞品：Tech Soft 3D — HOOPS AI

| 维度 | HOOPS AI | CAD ML Platform (我们) |
|------|----------|----------------------|
| **公司** | Tech Soft 3D（美国，CAD SDK 龙头） | — |
| **产品定位** | CAD 感知的 ML 开发框架/SDK | 完整的智能分析微服务平台 |
| **核心能力** | CAD 解析 + ML 流水线 + 嵌入向量 | 分类 + 特征 + 相似度 + 工艺 + 质量评估 |
| **格式支持** | 30+ 格式（HOOPS Exchange 引擎） | DXF, STEP, IGES（可扩展） |
| **交付形式** | SDK，需二次开发 | 独立微服务，API 即用 |
| **3D 分析** | 预建神经网络架构，B-Rep 支持 | UV-Net 双路径 GNN |
| **2D 图纸分析** | 有限 | **强项** — GNN + 48维特征 + 标题栏 OCR |
| **工艺推荐** | 无 | 有（基于规则 + ML） |
| **图纸质量评估** | 无 | 有 |
| **多租户/RBAC** | 无（SDK 不涉及） | 有（企业级 SaaS 架构） |
| **部署方式** | 嵌入客户应用 | Docker/K8s 独立部署 or 嵌入 |
| **平台** | Linux + Windows | Linux + Docker + K8s |
| **定价** | 未公开，按席位/运行时许可 | — |
| **成立时间** | HOOPS AI 2025年11月发布 | — |

**竞争评估**:
- HOOPS AI 是技术路线最接近的竞品，背靠 Tech Soft 3D 的 30+ 格式解析能力
- 但 HOOPS AI 是 **SDK 而非平台**，客户需要大量二次开发
- 我们在 2D DXF 分析、工艺推荐、质量评估、企业级特性方面有明显优势
- **威胁等级: 高** — 如果 HOOPS AI 向平台化演进，将构成直接竞争

## 七、战略建议

### 短期（0-6 个月）

1. **加速成本估算功能** — 直接对标 SIMILIA 和 simus classmate 的核心卖点
2. **LLM Copilot 差异化** — 市场上的 Copilot 都在"帮设计师画图"，我们做"帮工程师分析图纸"
3. **与国产 CAD 厂商合作** — 中望、浩辰、CAXA 作为集成伙伴，嵌入其生态

### 中期（6-12 个月）

4. **扩展 CAD 格式支持** — 接入 ODA 或 Open Cascade 扩展到 30+ 格式，缩小与 HOOPS AI 的差距
5. **构建零件库** — 与客户合作积累零件数据，形成数据飞轮
6. **图纸 Diff + ECN** — 市场独有能力，建立技术壁垒

### 长期（1-3 年）

7. **制造业知识图谱** — 材料+工艺+标准的结构化知识，结合 LLM 做 RAG
8. **联邦学习** — 多客户模型进化，不共享数据
9. **国际化** — 从中国制造业扩展到东南亚、欧洲中小企业
